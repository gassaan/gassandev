# Ismail Shareef — Personal Website

A single-page personal/professional website built from Ismail Shareef's CV: strategic
planning career at the Maldives Police Service, academic background, research
interests, and recognition.

## Structure

- `index.html` — page markup and content
- `assets/style.css` — styles (light/dark theme via CSS variables)
- `assets/script.js` — nav, theme toggle, scroll-reveal animations
- `assets/Ismail_Shareef_CV.pdf` — downloadable CV

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```
python3 -m http.server 8000
```
